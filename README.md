# 🔐 Secure Password Generator (Python)

A simple Python project that generates secure random passwords based on user-defined length.

## 🚀 Features

- Generate strong random passwords
- Includes:
  - Uppercase & lowercase letters
  - Numbers
  - Special characters
- User-defined password length
- Simple command-line interface
- Built using Python `random` and `string` modules

## 🧠 How It Works

1. User enters desired password length
2. Program combines letters, numbers, and symbols
3. Random characters are selected to form password
4. Final password is displayed with its length

## 🛠️ Technologies Used

- Python 3
- random module
- string module
